# Revision Notes

* OpenAPI describes REST API contract.
* Swagger UI displays interactive API documentation.
* Springdoc OpenAPI is commonly used with Spring Boot.
* Document request/response DTOs.
* Do not expose internal/admin docs publicly without security.
* Keep API documentation updated with implementation.

---

# Cheat Sheet

| Need | Tool |
| ---- | ---- |
| API spec | OpenAPI |
| UI | Swagger UI |
| Spring Boot lib | springdoc-openapi |
| Operation docs | `@Operation` |
| Schema docs | `@Schema` |

---

# Interview Questions with Answers

### 1. What is OpenAPI?

A standard specification for describing REST APIs.

### 2. What is Swagger UI?

Interactive UI generated from OpenAPI spec.

### 3. Why use OpenAPI?

It improves API discoverability, testing, and client integration.
