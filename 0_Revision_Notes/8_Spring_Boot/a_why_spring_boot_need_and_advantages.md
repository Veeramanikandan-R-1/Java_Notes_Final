# Interview Questions with Answers

### 1. Why Spring Boot?

It reduces setup effort and provides production-ready defaults for Spring applications.

### 2. What is starter?

A dependency bundle for a common capability like web, data JPA, or security.

### 3. Does Boot replace Spring?

No. It builds on top of Spring.
