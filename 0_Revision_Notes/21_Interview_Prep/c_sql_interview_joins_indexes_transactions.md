# Revision Notes

* SQL interviews focus on querying, performance, and consistency.
* Inner join returns matching rows.
* Left join returns all left rows.
* Indexes speed reads but slow writes.
* Transactions provide ACID guarantees.
* Execution plans help optimize queries.
* Avoid `SELECT *` in production queries.

# Interview Questions with Answers

### 1. What is left join?

It returns all rows from left table and matching rows from right table.

### 2. What is ACID?

Atomicity, Consistency, Isolation, Durability.
