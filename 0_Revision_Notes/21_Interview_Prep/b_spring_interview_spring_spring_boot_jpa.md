# Revision Notes

* Spring Core focuses on IoC and DI.
* Constructor injection is preferred.
* Spring Boot provides auto-configuration and starters.
* REST APIs should use DTOs.
* Validation and global exception handling are expected.
* JPA requires understanding entity states and transactions.
* N+1 problem is common interview topic.

---

# Cheat Sheet

| Area | Focus |
| ---- | ----- |
| Spring | IoC, DI, Beans |
| Boot | Auto config, starters |
| REST | Controller, DTO, validation |
| JPA | Entity, transaction, fetch |
| Production | Actuator, logs, profiles |

---

# Interview Questions with Answers

### 1. What is IoC?

Spring manages object creation and dependency wiring.

### 2. What is N+1?

Extra queries triggered for child data after loading parent records.
